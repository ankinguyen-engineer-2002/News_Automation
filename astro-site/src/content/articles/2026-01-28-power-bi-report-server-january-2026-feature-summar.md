Welcome to January 2026 Power BI Report Server feature summary! These updates bring new capabilities and enhancements across Reporting and Visualizations, maintaining our commitment to delivering powerful, flexible, and enterprise-ready reporting solutions.

For more information on [how to find your report server product key,](https://learn.microsoft.com/power-bi/report-server/find-product-key) refer to the documentation.

## Visualizations features

This release includes a variety of new and updated visuals from Microsoft partners and the broader community. These visuals enable richer storytelling, advanced exploration, and more engaging report experiences.

## Reporting features

This release also includes several reporting enhancements to improve layout behavior, visual styling, and interaction options to help you build polished, intuitive, and modern reports.

This concludes the feature summary for January 2026. Please follow this blog for updates on upcoming releases. As a reminder, Power BI Report Server releases occur in January, May, and September.

Thank you for using Power BI Report Server!
